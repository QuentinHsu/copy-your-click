## 0.0.3

### feat

## 0.0.2

### feat

- **Change Case**

	在资源管理器中右键点击文件，选择 `Copy As Case`，然后选择具体 `Case` 进行复制。
## 0.0.1

### feat

- **复制文件名**

	在资源管理器中右键点击文件，选择 `Copy File Name`，即可将文件名复制到剪贴板。

- **复制文件名（不带文件类型）**

	在资源管理器中右键点击文件，选择 `Copy File Name`，即可将不带文件后缀名的文件名复制到剪贴板。

- **复制文件夹名**

	在资源管理器中右键点击文件夹，选择 `Copy Folder Name`，即可将文件夹名复制到剪贴板。
